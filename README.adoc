== What is Cardi Buddy?
*Cardi Buddy* (CardiB) is for students who prefer to use a desktop app to store their revision materials as flashcards. If you can type fast, CardiB can help you to store revision materials quickly. The self-testing features will definitely help you to ace your examinations as opposed to traditional ways of revision by re-reading lecture notes.
